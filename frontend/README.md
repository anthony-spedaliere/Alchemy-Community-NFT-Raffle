# Frontend

This is the frontend application for the Alchemy Community NFT Raffle project.

For complete setup instructions, API documentation, and project overview, please refer to the main [README.md](../README.md) in the root directory.

## Quick Start

```bash
npm install
npm run dev
```

## Environment Setup

Copy `.env.example` to `.env.local` and fill in your configuration:

```env
ALCHEMY_API_KEY=your_api_key
CONTRACT_ADDRESS=deployed_contract_address
NEXT_PUBLIC_ADMIN_ADDRESS=admin_wallet_address
```</content>
